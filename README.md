# arg_example
Trying to replicate the examples from the paper 'Data-Driven First-Principles Methods for the Study and Design of Alkali Superionic Conductors"
- using pymatgen to prepare and analyze the VASP calculations
- DFT and AIMD calculation using VASP on Li6PS5Cl (argyrodite)

Link to paper: https://pubs.acs.org/doi/10.1021/acs.chemmater.6b02648
- see the SI for their Jupyter notebooks 
- or on matgenb: https://matgenb.materialsvirtuallab.org/
